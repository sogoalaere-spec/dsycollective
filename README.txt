DSY Collective Starter Website
---------------------------------
Minimalist and responsive design created for DSY Collective.

Sections:
- Hero (intro + CTA)
- About
- Lookbook
- Shop Preview
- Newsletter Form
- Footer (contact info)

How to Use:
1. Replace images in /assets.
2. Edit index.html for content.
3. Change colors/fonts in styles.css.
4. Preview locally by opening index.html.
5. Deploy to Netlify or Vercel (free).
